# Intellisense-Vehicle-based-on-stm32
基于stm32和五路数字量传感器的循迹小车,这是本人在大学期间参加的第一个校级比赛（我知道很菜别喷我），使用了五路灰度传感器感知赛道信息，stm32f103c8t6作为主控芯片。本程序未使用pid控制小车速度，未使用编码器获取当前速度。
